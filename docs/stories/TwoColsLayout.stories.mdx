# TwoColsLayout

`TwoColsLayout` creates a responsive 2 column grid.

[View source](https://github.com/strapi/design-system/tree/main/packages/strapi-design-system/src/TwoColsLayout)

## Imports

```js
import { TwoColsLayout } from '@strapi/design-system';
```
