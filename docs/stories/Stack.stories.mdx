## Usage

Add vertical spacing between direct child: it allows not to have to deal with `margin-top` and `margin-bottom` anymore.

## Props

The Stack component wraps all its children in the
(Box)[https://design-system-git-main-strapijs.vercel.app/?path=/docs/design-system-technical-components-box--base]
component, so you can pass all Box props to change its style.
