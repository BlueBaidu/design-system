# Link

Links are a way to navigate to different pages whether internal or external.

Best practices

- A link should have a relevant and clear label.
- Be positioned in consistent locations in the interface.
- Add an icon next to the link to emphasize the required action.

### Disabled link

Depending on the permissions a user have or the status of an action, a link can be unreachable or unavailable yet.
